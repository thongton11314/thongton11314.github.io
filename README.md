## Getting Started

1. Clone this repository
2. Open `index.html` in your browser (or use VS Code Live Server)
3. Search for `[` to find all placeholder text and replace with your info

## Deployment

Push to a GitHub repository named `yourusername.github.io` and enable GitHub Pages in Settings → Pages → Deploy from branch `main`.
